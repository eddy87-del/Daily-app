# Daily Milk App

A simple web application to track daily milk consumption and manage milk-related activities.

## Features

- 📊 Track daily milk consumption
- 📈 View consumption history and statistics
- 📱 User-friendly interface
- 💾 Local data persistence
- 📅 Daily logging and tracking

## Project Structure

```
Daily-app/
├── index.html          # Main HTML file
├── styles/
│   └── style.css       # Application styles
├── scripts/
│   └── app.js          # Main application logic
├── README.md           # This file
└── package.json        # Project metadata
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Start tracking your daily milk consumption

## Usage

- Enter the amount of milk consumed (in ml or oz)
- Select the time of day
- Click "Log" to save the entry
- View your daily statistics and history

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API
